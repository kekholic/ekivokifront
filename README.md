# Ekivokifront
Online game Ekivoki for 2-4 players!

# Overview
This game was build using next technologies:

+ TypeScript
+ React
+ Redux
+ Express
+ Socket.io
+ Canvas
+ JWT

# Installation

1. Don't forget about back part! https://github.com/kekholic/ekivokiback
2. `npm i`
3. `npm start`
